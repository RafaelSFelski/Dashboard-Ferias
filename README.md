# Dashboard de Controle de Férias

Um sistema completo para gerenciamento de férias de funcionários.

## Funcionalidades

- 📊 **Dashboard com estatísticas** em tempo real
- ➕ **Adicionar/Editar** funcionários
- 🔍 **Busca** por nome ou departamento
- 🏷️ **Filtros** por status (Disponível, Programado, Em Férias, Concluído)
- 💾 **Armazenamento local** dos dados (localStorage)
- 📱 **Design responsivo** para mobile e desktop
- 🎨 **Interface moderna** com gradientes e animações

## Como usar

1. Abra o arquivo `index.html` em seu navegador
2. Use o formulário para adicionar novos funcionários
3. Visualize as estatísticas no topo da página
4. Filtre e busque funcionários na tabela
5. Edite ou exclua registros conforme necessário

## Tecnologias

- HTML5
- CSS3 (com Flexbox e Grid)
- JavaScript (Vanilla)
- LocalStorage para persistência de dados

## Estrutura de Arquivos

- `index.html` - Estrutura da página
- `styles.css` - Estilização e design
- `script.js` - Lógica e funcionalidades
